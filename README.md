# Windows Quick Start (No Terminal)

## 1) Install AutoHotkey v2
- https://www.autohotkey.com/

## 2) Get files
- Download ZIP from: `https://github.com/Samefisk/prompts`
- Extract and open `windows` folder

## 3) First-time setup
- Double-click `setup_ct.ahk`
- Paste your Gemini API key
- Keep defaults unless instructed otherwise

## 4) Use it
- Double-click `ct.ahk`
- In any text field, type `,ct`

## Files that must stay together
- `ct.ahk`
- `ct.ps1`
- `setup_ct.ahk`
- `ct-config.json` (created by setup)

## Default settings
- Prompt URL: `https://raw.githubusercontent.com/Samefisk/prompts/main/super-prompt.md`
- Model: `gemini-3-pro-preview`
- Thinking: `LOW`
